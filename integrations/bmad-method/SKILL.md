# BMAD-METHOD Integration

Build More Architect Dreams — an AI-driven agile development framework with 12+ specialized agent personas and scale-adaptive intelligence that adjusts from bug fixes to enterprise systems.

## What It Adds

- **12+ specialized agents** — PM, Architect, Developer, UX Designer, and more
- **Scale-adaptive planning** — automatically adjusts depth based on project complexity
- **Party Mode** — multiple agent personas collaborating in one session
- **Module ecosystem** — extend with Test Architect, Game Dev, Creative Intelligence

## How It Complements Spec-Harness

| Spec-Harness | BMAD | Use Together When |
|-------------|------|-------------------|
| /spec-author | BMAD PRD skill | BMAD's PM agent can write richer product briefs |
| /implementer | BMAD Developer agent | BMAD agents follow Spec-Harness TDD rules |
| /reviewer | BMAD Architect agent | Architect reviews design depth before code review |
| /grill-me | BMAD brainstorming | BMAD's scale-adaptive depth adjusts automatically |

## Setup

```bash
npx bmad-method install
```

Follow the installer prompts to select modules and target IDE.

## Usage Pattern

1. Use BMAD's **PM agent** for initial product brief and user stories
2. Transition to Spec-Harness' `/spec-author` for EARS/Gherkin formalization
3. Use BMAD's **Architect agent** for deep design reviews (before `/implementer`)
4. Use Spec-Harness' `/reviewer` and `/mutation-tester` for code-level verification
5. Use BMAD's **`bmad-help`** anytime for guidance on what's next
